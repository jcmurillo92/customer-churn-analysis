# Tableau Dashboard

This folder contains Tableau-related assets for the customer churn analysis project.

## Dashboard Purpose
The Tableau dashboard was built to visualize:
- overall churn rate
- churn by contract type
- churn by customer tenure
- churn by payment method

## Main Insights Visualized
- Month-to-month customers had the highest churn rate
- Customers in their first 12 months had the highest churn risk
- Electronic check customers had the highest churn rate among payment methods
